# きたくグラフ

Twitter で @yono が「きたく」とつぶやいた時刻をグラフにして表示します。

## Requirements

- RVM
- Ruby 1.9.3
- RubyGems
  - bundler

## Install

Install Ruby.

    $ rvm install 1.9.3
    $ rvm use 1.9.3
    $ rvm gemset create kitakugraph
    $ rvm gemset use kitakugraph

Install gems.

    $ bundle install

## Usage

    $ ruby app.rb
